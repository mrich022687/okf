## Headless Conversion (Completed 2026-06-29)

### What was done:
1. **Removed XFCE desktop + LightDM:**
   - `apt purge xfce4* lightdm*` — cleared ~2 GB of unnecessary packages
2. **Disabled junk services permanently:**
   - `bluetooth`, `cups`, `ModemManager`, `accounts-daemon`, `avahi-daemon`, `colord`
3. **Removed Firefox** (was consuming 2+ GB RAM with 72+ tabs)
4. **Disabled Plymouth splash** — boots to text console instead of Mint logo
   - `GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"` → `GRUB_CMDLINE_LINUX_DEFAULT=""`
   - Ran `update-grub`

### What the screen shows at boot:
After POST, boot messages scroll by, then:
```
[  boot messages...  ]
q1900m login: _
```
No Mint logo, no GUI. Just a terminal login prompt.

### What was NOT done (future):
- SSH key auth (passwordless login) — still on password
- Credential vault setup — still plain passwords

### How to reboot:
```bash
sshpass -p 'mcrart8794' ssh michael@192.168.12.208 "sudo reboot"
```
Shares (NFS/Samba) drop for ~1 minute while it comes back.
